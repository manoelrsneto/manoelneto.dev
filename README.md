# manoelneto.dev

Personal blog. Bilingual (EN/PT), terminal aesthetic, statically generated.

## Stack

- [Astro](https://astro.build) — static site generator
- [MDX](https://mdxjs.com) — post authoring
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Pagefind](https://pagefind.app) — static full-text search
- [D2](https://d2lang.com) — diagrams

## Quick start

```bash
pnpm install
pnpm dev      # http://localhost:4321
pnpm check    # lint + types before committing
```
